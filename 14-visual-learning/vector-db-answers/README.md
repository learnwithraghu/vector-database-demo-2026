# Vector Quest: How Semantic Search Works

This interactive lab guides students step-by-step through the journey of a query inside a Vector Database. 

Students select a question and watch it travel through the core processes of semantic search:
1. **Tokenisation**: Breaking raw text into machine-readable sub-word units.
2. **Embedding**: Transforming tokens into a dense, high-dimensional vector that captures meaning.
3. **Vector Space**: Visualizing nearest-neighbour search using cosine similarity to find documents with closely aligned vectors.
4. **Results**: Seeing how meaning-based retrieval succeeds without relying on exact keyword matches.

Instructors can use this walkthrough to demystify the "magic" of vector databases and concretely explain how semantic context is mathematically operationalized.
