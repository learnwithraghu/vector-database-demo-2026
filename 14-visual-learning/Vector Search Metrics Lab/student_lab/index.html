Student lab · Vector search

Similarity metrics & animal embeddings

Each arrow is a fake 2D “embedding” for an animal. The orange arrow is your query. Switch metrics and read the ranked list — then answer the multiple choice questions from what you see.

You will work through four short stages and a summary:

1 — Euclidean distance (closest tip in the plane)

2 — Cosine similarity (direction)

3 — Dot product (direction × length)

4 — Compare metrics after dragging the query

No code to edit — only the interactive UI.

Stage 1 — Euclidean distance

Euclidean distance is the straight-line gap between the query tip and each document tip. Smaller distance ⇒ higher rank. Use the ranking table when answering.

Embedding plane

This stage uses Euclidean distance only. The query (orange q) is fixed.

Dashed lines show straight-line distance from the query tip to each animal tip.

Ranked results

Recap: Euclidean answers “how far apart are the tips in this 2D map?”

Stage 2 — Cosine similarity

Cosine compares direction from the origin, not raw distance. A document can be farther away yet more “aligned” with the query.

Embedding plane

This stage uses cosine similarity (direction from the origin). Same animal vectors as before.

Use the table: rank #1 has the highest cosine score.

Ranked results

Recap: Cosine is about the angle between vectors — like pointing the same way.

Stage 3 — Dot product

Dot product grows when vectors align and when they are long. Rankings can differ from both Euclidean and cosine.

Embedding plane

This stage uses dot product (combines alignment and vector length).

You may need row #2 in the table for the question — not always #1.

Ranked results

Recap: Dot product mixes alignment and magnitude.

Stage 4 — Compare the metrics

All three metrics are available. Drag the query to a new spot, switch metrics, and watch #1 change (or not). Then answer using the live table.

Embedding plane

Drag the orange query tip. Switch metric and compare top ranks — the question uses dot vs Euclidean.

Rankings update as you drag. Use the table after switching each metric.

Ranked results

Recap: “Nearest neighbor” depends on the metric you choose.

Summary

What you practiced

  • Euclidean — distance between arrow tips in the plane.
  • Cosine — how well directions match, ignoring length in the formula.
  • Dot product — alignment and vector length together.
  • Same animal vectors + same query can yield different #1 when the metric changes.
