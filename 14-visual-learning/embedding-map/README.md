# The Embedding Map

This interactive lab visually demonstrates how word embeddings map meaning into a vector space. 

Students learn that embeddings group words by their semantic relationships rather than superficial traits like spelling or length. 
- **Level 1** introduces spatial clustering, showing how related concepts naturally gather together on a 2D map. 
- **Level 2** introduces "Semantic Mathematics" (e.g., `Car - Motor + Pedals = Bicycle`), allowing students to see how adding and subtracting vectors logically manipulates conceptual meaning.

Instructors can use this lab to build an intuitive, visual foundation for embeddings before diving into the mathematical details of vector dimensions.
