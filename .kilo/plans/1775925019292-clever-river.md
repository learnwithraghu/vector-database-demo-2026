## Implementation Plan

### Stage Breakdown (6 stages, no-scroll)

**Stage 0: Hook**
- Headline: "The AI gave a confident answer. It was completely wrong."
- Subtext: "Here's why — and how the smartest AI systems fix it."
- Animated brain icon: question mark → checkmark morph (CSS keyframes)
- Button: "Show me what's wrong →"
- No progress dots

**Stage 1: Closed-Book Problem**
- Concept label: "The problem: AI answers from memory alone"
- Left: Chat UI with Acme Corp question + typewriter invented answer
- Right: Closed-book exam illustration
- Warning banner slides up after typewriter
- Unlock: Auto after ~4 seconds

**Stage 2: Knowledge Vault**
- Concept label: "The fix: give the AI a vault of real documents"
- Vault container with 4 chunk cards fanned out
- Click "Search the vault →" → Chunk A glows/highlights
- Unlock: After search completes

**Stage 3: Handoff (Key Interaction)**
- Concept label: "Now: hand the AI the right page"
- Split 3 zones: Chunk left, pulsing arrow center, AI right
- Click "Inject context →" → Arrow animates, AI wakes up, types grounded answer
- Green badge: "✓ Grounded. Every fact traced to the source."
- Unlock: After answer types out

**Stage 4: What Just Happened**
- Concept label: "What you just did has a name: RAG Retrieval-Augmented Generation"
- Pipeline animation: 5 nodes with flowing dashed line
- Unlock: Auto after ~3 seconds

**Stage 5: Recap**
- Headline: "You just taught an AI to stop guessing."
- 3 takeaway cards with icons, staggered fade
- Closing line about all AI assistants using this
- Two buttons: "Explore Again" (restart) and "What's a vector DB?" (sendPrompt)

### Technical Implementation

**Fonts & Typography**
- Import Syne (700) and DM Sans (400/500/600) from Google Fonts
- Strict font sizes:
  - Stage headline: 48px Syne 700
  - Concept label: 24px Syne 700 accent
  - Body: 18px DM Sans 400
  - Chunk text: 17px DM Sans 400
  - Node labels: 15px DM Sans 500
  - Helper: 16px DM Sans 400 muted
  - Buttons: 17px DM Sans 600

**Animations & Interactions**
- Brain morph: CSS keyframes morphing question mark to checkmark
- Typewriter: Character-by-character with setInterval
- Search pulse: Ripple effect through vault, Chunk A highlight
- Arrow flow: Glowing arrow animation on inject
- Pipeline: Sequential node fade-in with dashed connecting line
- Warning banner: Slide up from bottom
- Stage transitions: Opacity + translateX slide

**Layout & Spacing**
- All stages: position absolute, inset 0, flex center
- Padding: 0 8vw on all stages
- Generous gaps: min 2rem between elements
- Next button: Fixed bottom center, pill shape, accent bg

**Code Structure**
- Single HTML file, no external dependencies
- CSS variables for Dark Cosmic palette
- Vanilla JS for all interactions
- Progress dots from Stage 1 onward
- Clean timer/animation cancellation on navigation
